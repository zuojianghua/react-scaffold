# 说明

## serverless部署工具

```npm install serverless -g```

### `serverless deploy`

部署到腾讯云

### serverless配置

./serverless.yml

### 自动构建配置

./.github/workflows/node.js.yml

## 脚手架

### `npx create-react-app myapp --template typescript`

初始化react应用ts版本

### `yarn add antd`

引入antd

### `yarn add react-router-dom`

引入react路由

### `yarn start`

Open [http://localhost:3000](http://localhost:3000)

### `yarn build`
